# Grundlagen der Linux Shell

Shell = textorientierte Schnittstelle zu einem Betriebssystem (Kernel)

Anzeige in der Konsole: Prompt
Eingabe nach dem Prompt

Beispiel: Der Befehl `date`

```sh
$ date
```

Ausgabe:

```
Wed Sep 24 07:11:30 AM UTC 2025
```

Befehle können Optionen haben, diese werden mit `-` angeben.

z.B.

```sh
$ date -I
```

Ausgabe:

```
2025-09-24
```

Optionen aus einem Buchstaben = shorthand

longhand mit 2x `--`

```sh
$ date --iso-8601
```

Mehr Informationen zu einem Befehl:

```sh
$ date --help
```

- man pages

```sh
$ man date
```
nicht in Replit aber man kann mit dem TExt `man date`im Internet suchen.
