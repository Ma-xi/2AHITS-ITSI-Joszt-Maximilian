## 1. Schrittt Replit mit Github verbinden 

In Repplit ein Fenster mit dem Titel `Git` öffnen dort in die Einstellungen gehen (kleines Zahnrad).
Dort neben dem Github auf anmelden klicken und sich mit seinem Account anmelden.

## 2. Schritt Github vorbereiten

In Github ein `Reposetory` anlegen mit dem folgenden Namen: <Klasse>-<Fach>-<Nachname>-<Vorname>.
Außerdem den Link des Reposetories kopieren und in den Repplit Einstelungen bei `Remote`einfügen.

## 3. Datein ins Github bringen 

Unter den ganzen Änderungen auf den Knopf drücken wo `Stage und commit all changes` steht.
Danch ganz oben auf Sync changes gehen damit alles im Github landet.

## 4. Die Webseite erstellen lassen

In den Einstellungen des reposetories auf `Pages` gehen und dort unter `Branch` `main` auswählen und auf save drücken.
Dann wird unter `Actions` in ein Workflow gestartet der etwas zeit beanspruchen kann.
Wenn dies erledigt drückt auf den Workflow klicken und dort im Diagramm unter `deploy` den Link öffnen.

## 5. Die Einzelnen Berichte anzeigen lassen 

Auf der Webseite ist bjetzt nur die `README` Datei um auf die einzelnen Berichte zu kommen muss man in Github unter `Code` auf den Bericht drücken und ganz oben den Link kopieren.
Das Kopierte hinter die `URL` der Webseite schreiben und das `.md` zu `.html` ändern oder es einfach löschen.

## 6. README verändern

Außerdem ist es möglich auf der README Links zu den Berichten zu erstellen dies kann man im Replit so erledigen:
```
[Text auf den man klickt](Link aus dem kopierten Bericht)
```
