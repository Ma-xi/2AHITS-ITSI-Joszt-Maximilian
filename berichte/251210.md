## wget

Mit `wget` kann man Webseiten runterladen und sie direkt in eine Datei speichern.

Eingabe:

```shell
wget https://www.franzmatejka.at/htl/doc/ITSI_2_linux/testdata/shopping.txt

```

Die Url endet in diesem Fall mit `.txt` also wird die Webseite in eine Textdatei gespeichert.

## Head

Es gibt ja den Befehl `cat` mit diesem kann man den ganzen Inhalt einer Datei herauslesen
mit `Head` dagegen werden nur die ersten 10 Zeilen ausgelsen.

Eingabe:

```shell
head shopping.txt
```

Ausgabe:

```shell
Fred apples 20
Susy oranges 5
Mark watermellons 12
Robert pears 4
Terry oranges 9
Lisa peaches 7
Susy oranges 12
Mark grapes 39
Anne mangoes 7
Greg pineapples 3

```

Es gibt noch die Funktion `-n` damit kann man bestimmen wie viele Zeilen ausgelsen werden sollen die Longhand form wäre `--lines=`.

Eingabe:

```shell
head -n3 shopping.txt
```

Ausgabe:

```shell
Fred apples 20
Susy oranges 5
Mark watermellons 12
```

## seq

Seq erstellt einfach eine Zahlenfolge von 1 bis zur angegebenen Zahl.

Eingabe:

```shell
seq 5
```

Ausgabe:

```shell
1
2
3
4
5
```

---

## nl

`Nl` nummeriert Zeilen aus einer Datei und gibt sie mit nummern davor wieder aus.

Eingabe:

```shell
nl test.txt
```

Ausgabe:

```shell
1 Hallo
2 Welt
```

Aufgabe: Was passiert wenn bei head eine negative Zahl für die Anzahl der Zeilen verwendet wird? Was steht dazu in der manpage von head?

Lös:

Eingabe:

```shell
head -n-5 shopping.txt
```

Wenn man ein Minus vor die Zahl schreibt werden alle ausgegeben bis auf die letzten Zeilen.
