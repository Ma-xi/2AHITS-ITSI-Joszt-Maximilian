## Cat 

Der Befehl `Cat` kann Dateien erstellen in diese etwas reinschreiben und zusammenfügen.

Es gibt `Stdin` das ist der `Standartinput` dieser ist standartmäßig die Tastatur.

Es gibt auch noch `Stdout` das ist der `Standartoutput` dieser ist standartmäßig der Bildschirm.

### Bsp:

Eingabe:

```sh
cat 
xyz
asdf
bnmk
```

Ausgabe:

```sh
xyz
asdf
bnmk
```

Hier gibt `cat` die durch die Standarteingabe eingegebenen Worte wieder die gleichen Worte bei der Standartausgabe aus.


```sh
cat >test.txt

xyz 
abcd
efgh
```

Hier wird die Datei test erstellt wenn diese noch nicht existiert und dann die Worte in diese geschrieben.

Wenn schon etwas darin vorhanden war wird dies mit dem neuen überschrieben dies kann man durch `>>` verhindern.

Wenn ein Sonderzeichen eingegeben werden soll brauch man diese Zeichen `''` davor und dahinter.

```sh
cat <test.txt

```

Hier wird dann ausgegeben was in der Datei steht.

### Übung (os-release)

Es gibt eine Datei /etc/os-release.

Betrachte deren Inhalt mit cat, welche Informationen kannst du daraus ableiten. Würde diese Information einem Hacker helfen der sich Zugriff auf das System verschafft hat? Wenn ja, wie?
Wie wird die Datei mit ls -l angezeigt? Was bedeutet diese Anzeige?

Man kann dort herauslesen `welches Linux Version das System hat`, `die Versionsnummer` und den `Codename` aber natürlich auch noch mehr.

Ja es kann einem Hacker helfen aber nicht besonders viel es hilft ihm also nur ihm sich darin zu orientieren.

Mit ls wird es so angezeigt:

```shell
lrwxrwxrwx 1 root root 21 Feb  5  2025 /etc/os-release -> ../usr/lib/os-release

```


### Übung (Textdatei erstellen)

Mit Hilfe von mehreren echo Befehlen können einfache Textdateien erstellt werden.

Erzeuge nur unter Verwendung von echo Befehlen die Datei made_by_echoing.txt mit Inhalt:

=====================
=    HTL BRAUNAU    =
=====================
= 2AHITS Gruppe n   =
= x Schülerinnen    =
= y Schüler         =
=====================
Setze für n, x und y die richtigen Werte ein.


Eingabe:

```shell
echo "=====================" > made_by_echoing.txt 
echo "=    HTL BRAUNAU    =" >> made_by_echoing.txt
echo "=====================" >> made_by_echoing.txt 
echo "= 2AHITS Gruppe 1   =" >> made_by_echoing.txt 
echo "= 0 Schülerinnen    =" >> made_by_echoing.txt 
echo "=12 Schüler         =" >> made_by_echoing.txt 
echo "=====================" >> made_by_echoing.txt


```

Ausgabe:

```shell

cat < made_by_echoing_txt

=====================
=    HTL BRAUNAU    =
=====================
= 2AHITS Gruppe 1   =
= 0 Schülerinnen    =
=12 Schüler         =
=====================

```
